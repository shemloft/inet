Сканер TCP и UDP портов
Определяет какие порты открыты на заданном хосте в заданном диапазоне, а также какие протоколы работают на открытом порте. Умеет распознавать http, smtp, pop3, dns, sntp

Пример запуска:
python port_scanner.py 127.0.0.1 50 55 -s udp

Справка: 
python port_scanner.py -h

Если не указан аргумент "-s udp" сканируются tcp порты
